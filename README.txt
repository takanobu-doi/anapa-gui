GUIでイベントと端点のタグ付けをするプログラムを作る。
rootのGUI機能を使って処理をする。
イベントのタグ付けをボタンを使って処理、
端点はクリックで処理。
解析する人のユーザー名をリストボックスから選んで、
ユーザーごとに処理をした結果をテキスト形式で保存していく。
解析に必要なファイルをプログラムの引数に指定するときには
ファイル名を指定すると間違える原因になるから
ラン番号で指定するか、プログラムを実行した後にプルダウンで指定する。

必要なボタンとテキストボックスの種類
*ユーザー関係
-新規のユーザー名を入力するテキストボックス
-新規のユーザーをaddするためのボタン
-ユーザーのリストを選ぶボックス
*解析関係
-イベントの種類ごとのボタン (ラジオボタン)
-解析開始ボタン (このボタンを押すことでイベントの種類の入力もする)
-選んだイベントの種類に応じた数だけクリックをさせる
-可能なら"次へ"のウィンドウをポップアップ？

出力・読み込みをするファイルの形式
*ユーザーリスト
-ユーザー名
-ユーザーID
-進行状況
-ログイン状態
*解析結果 (ユーザーごとに生成)
-RUN No.
-Tree Event No.
-イベントの種類
-端点の座標 ax ay cx cy (pixel) で端点の数だけ
